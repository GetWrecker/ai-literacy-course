1. Context | 2. Role | 3. Examples | 4. Audience | 5. Delivery

Bad Prompt: "Write a post about AI."

C.R.E.A.D Prompt: "I am launching a tech blog (Context). Act as a senior tech editor (Role). Use short sentences like Wired Magazine (Examples) for beginner developers (Audience). Format as Markdown (Delivery)."
