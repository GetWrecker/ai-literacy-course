# 🤖 Interactive AI Literacy Course

An all-in-one, open-source curriculum designed to bridge the gap between AI theory and practical implementation. This course helps individuals and businesses understand the foundations, navigate the risks, and implement AI tools like Chatbots and GEO.

---

## 🌟 Key Features
* **Modular Learning:** 5 comprehensive modules covering everything from basic definitions to business integration.
* **GEO Strategy:** Learn **Generative Engine Optimization** to make your website "AI-ready."
* **Interactive Components:** Built-in sentiment analysis tool and knowledge checks.
* **Modern UI:** Clean, responsive design built with HTML5 and CSS3.
* **Lightweight:** No heavy frameworks; runs directly in any browser.

---

## 📚 Course Curriculum

1.  **Module 1: What is AI?** – Understanding the core logic and importance.
2.  **Module 2: Risks & Ethics** – Deepfakes, bias, and data privacy.
3.  **Module 3: Daily Life** – Practical AI hacks for personal productivity.
4.  **Module 4: Business & GEO** – Scaling businesses and optimizing for AI search.
5.  **Module 5: Chatbot Integration** – Building and deploying virtual assistants.
